Usage
---

1. Install `npm` dependencies:
  ```sh
    npm install -g webpack webpack-dev-server
    npm install
  ```

2. Start `webpack-dev-server` with live reload (*Note: `.html` files are not watched*):
  ```sh
    npm start
  ```

3. Bunle with `webpack`:
  ```sh
    webpack
  ```
